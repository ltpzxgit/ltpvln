# ltp's Linux & DevOps Cheatsheet

A collection of practical Linux, Docker, Git, and DevOps commands and scripts for daily use.
Organized for quick reference and ready to be used as a starting point for a personal GitHub repository.

> Minimal, clean, and practical.

## Structure
- `linux/` — Linux essentials
- `docker/` — Docker & Compose
- `git/` — Git workflow
- `scripts/` — Automation scripts (bash)
- `devops/` — Tools like SSH, tmux, etc.
